# KosmosUtils
> Kosmos的工具库,因为我比较懒，所以折腾了这个东西，
> 把我平时在用的工具们抽取了出来
> 方便以后直接include进来

---
#### 其中：
`ULog`:是基于[Blankj](https://github.com/Blankj/AndroidUtilCode)的工具封装

`UToast`:是基于鸿洋[ToastUtils](https://github.com/getActivity/ToastUtils)封装
